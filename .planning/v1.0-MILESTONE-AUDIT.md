### Integration Gap: INV-02 Slot-Full Rejection

**File:** `scripts/resources/inventory.gd` lines 55–58
**Issue:** `insert_rejected` only emitted when `weight_budget <= 0`. When all 15 slots are occupied but weight capacity remains, `insert()` fails silently.
**Requirement:** INV-02 states rejection message shown when inventory is "full or overweight".
**Fix scope:** Single branch addition in `inventory.gd:insert()`. HUD wiring is already in place.
